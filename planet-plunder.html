Planet Plunder

An action-exploration game featuring procedurally generated planets, a branching player skill tree, and dynamic resource gathering.

C# Unity Procedural Generation Skill Tree
01. Overview

What is Planet Plunder?

Planet Plunder is a game where players explore procedurally generated planets in search of valuable resources, ancient artifacts, and hidden challenges. Each planet is uniquely constructed at runtime, meaning no two expeditions are ever the same.

The core gameplay loop revolves around landing on a planet, navigating its terrain, gathering materials, and investing earned progress into a deep skill tree system that unlocks new abilities, upgrades, and exploration strategies.

The game is currently under active development as a personal project, showcasing my skills in gameplay programming, procedural content generation, and systems architecture within Unity.

02. Core Feature

Procedural Generation

Every Planet Tells a Different Story

The procedural generation system creates unique planetary surfaces by layering noise algorithms, biome distribution rules, and resource placement logic. Terrain elevation, vegetation density, hazard zones, and points of interest are all determined algorithmically at runtime.

  • Multi-layered noise-based terrain generation
  • Dynamic biome placement with smooth transitions
  • Randomized resource scattering and loot tables
  • Seed-based reproducibility for shared planet codes
Procedurally generated terrain with biome variations
03. Core Feature

Skill Tree System

Shape Your Playstyle

The skill tree provides meaningful progression choices. As players complete expeditions and earn experience, they unlock nodes across multiple branches — each focusing on different gameplay pillars like traversal, combat efficiency, resource detection, and survival.

  • Multiple branching paths with distinct specializations
  • Active and passive abilities unlocked through progression
  • Node dependencies and prerequisites for strategic depth
  • Visual node graph rendered in-game with real-time feedback
