C# Unity Gameplay Physics Interactive Sandbox
01. Overview

Character Casting Engine

In action RPGs and magic-based games, the player controller must feel incredibly fluid and allow complex physics integrations for spell casting. This project showcases a custom C# Unity player controller that handles complex spell physics, casting states, and modular spell definitions.

The core architecture uses a decoupling methodology: the Player Controller handles locomotion, state machines, and input processing, while a modular Spell Casting Component manages casting states, cooldowns, and instantiates customizable physical spells.

To demonstrate how casting variables affect gameplay feel and physics logic, you can interact with the **Spell Sandbox Simulation** below. Change casting properties and cast spells dynamically!
