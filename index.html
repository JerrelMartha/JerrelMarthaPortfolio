About Jerrel Martha

Architecting fluid gameplay mechanics with robust, clean game systems.

I am a passionate game programmer specializing in the creation of immersive gameplay features, editor tools, and core subsystems. My work spans from micro-interactions on the controller to the foundational frameworks that power character physics, AI behaviors, and item systems.

Currently, I am expanding my technical foundations as a student at **Grafische Lyceum Utrecht (GLU)** (2024 - Present), focusing on game programming methodologies, asset integration pipelines, and interactive game architectures.

I focus on writing high-performance, maintainable C# and modern web languages, with plans to learn C++ to expand my systems-programming capabilities. I believe in giving game designers ultimate control by crafting custom editor utilities and tools that streamline content pipeline creation and accelerate development iteration.
